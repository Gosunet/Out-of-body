Prérequis : 

- Effectuer l'installation de node.js et exécuter "npm install" dans le dossier ApplicationMenu


- Lancer la commande "npm install -g grunt-cli" 



Pour Développer

- Exécuter "npm install" dans le dossier ApplicationMenu

- Lancer la commande "node server.js" puis dans un autre terminale node : "grunt"
